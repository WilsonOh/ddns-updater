# Supported Platforms

1. MacOS
2. Linux

# Getting Started

1. `git clone https://github.com/WilsonOh/ddns-updater`
2. `make build`
3. `make init_config`
4. Update config values (cloudflare email, api token, domains and subdomains etc.) in ~/.config/ddns-updater/config.json
5. `./bin/ddns-updater`
